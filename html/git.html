Git

  • ls-Показать файлы в данной папке, кроме скрытых.
  • cd - Для навигации по файлам и каталогам в системе используйте команду cd, за которой следует полный путь или имя каталога. Запуск этой команды без флага или параметра перенесет вас в домашнюю папку.
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • mkdir - расшифровывается как "Make directory", что переводится как "создать папку".
  • mv app1/*.* app2 - расфшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно перемещать файлы между директориями /
